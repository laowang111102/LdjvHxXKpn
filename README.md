# 音乐推荐系统 python819

## 项目概述

本项目是一个基于Python协同过滤算法的音乐推荐系统，旨在为用户提供个性化的音乐推荐服务。系统包括源码、数据库以及详尽的文档说明。

## 技术栈

- **后端**: Python + Django
- **数据库**: MySQL
- **前端**: Vue.js
- **核心算法**: 协同过滤算法

## 功能模块

### 管理员功能

- 用户管理
- 音乐分类管理
- 音乐信息管理
- 歌曲数据管理
- 系统管理
- 我的信息管理

### 用户功能

- 个人中心
- 修改密码
- 我的收藏

## 系统角色

- 管理员：负责管理和维护整个系统的运行。
- 用户：享受音乐推荐服务，管理个人账户信息。

## 运行环境

- 后端：Python 3.x，Django框架
- 数据库：MySQL
- 前端：Vue.js，支持现代浏览器

## 部署步骤

1. 环境准备：安装Python、MySQL、Vue.js等必要软件。
2. 数据库搭建：根据项目文档建立数据库和表结构。
3. 后端部署：配置Django项目，准备运行环境，部署后端代码。
4. 前端部署：编译Vue.js项目，部署至Web服务器。
